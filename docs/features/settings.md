# 功能设计：设置与个性化

## 功能概述

设置页面允许用户自定义插件的行为、外观和数据管理策略。所有设置本地持久化，即时生效（无需重启插件）。

---

## 设置分类

### 1. 阅读体验

| 设置项 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| 主题 | 单选 | 跟随系统 | 浅色 / 深色 / 跟随系统 |
| 正文字号 | 单选 | 中 | 小（14px）/ 中（16px）/ 大（18px） |
| 正文字体 | 单选 | 衬线 | 衬线 / 无衬线 / 系统默认 |
| 打开即已读 | 开关 | 关 | 点击文章是否立即标记已读 |
| 滚动自动已读 | 开关 | 开 | 滚动超过 80% 自动标记已读 |
| 默认显示 | 单选 | 全部 | 默认展示"全部"还是"仅未读" |

---

### 2. 刷新设置

| 设置项 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| 自动刷新间隔 | 单选 | 1 小时 | 15分钟 / 1小时 / 4小时 / 禁用 |
| 刷新时显示通知 | 开关 | 关 | 有新文章时弹出系统通知 |

---

### 3. 数据管理

| 设置项 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| 每 Feed 最大文章数 | 单选 | 200 | 50 / 100 / 200 / 500 / 不限制 |
| 保留已读文章天数 | 单选 | 30 天 | 7天 / 30天 / 90天 / 永久 |
| 导入 OPML | 操作 | — | 选择文件导入 |
| 导出 OPML | 操作 | — | 下载订阅列表 |
| 清空文章缓存 | 危险操作 | — | 删除所有文章，保留订阅 |
| 重置插件 | 危险操作 | — | 清空所有数据 |

---

### 4. 快捷键

| 快捷键 | 功能 | 是否可自定义 |
|---|---|---|
| `/` | 聚焦搜索框 | ❌ |
| `j` | 下一篇 | ✅ |
| `k` | 上一篇 | ✅ |
| `o` | 打开原文 | ✅ |
| `m` | 切换已读 | ✅ |
| `s` | 收藏 | ✅ |
| `r` | 刷新当前 Feed | ✅ |
| `Esc` | 返回列表 | ❌ |

可自定义的快捷键支持用户在设置页面重新绑定单字母键（不支持组合键，避免与浏览器冲突）。

## 设置持久化

设置数据存储在 IndexedDB 的 `settings` 表中（单行，`id = 'global'`）：

```typescript
// 读取设置
async function getSettings(): Promise<Settings> {
  const settings = await db.settings.get('global');
  return settings ?? DEFAULT_SETTINGS;
}

// 更新设置（部分更新）
async function updateSettings(patch: Partial<Settings>): Promise<void> {
  await db.settings.update('global', patch);
  // 广播给 Background Service Worker
  chrome.runtime.sendMessage({
    action: 'SETTINGS_UPDATE',
    payload: patch
  });
}
```

## UI 设计

设置页面通过 Popup 内的导航进入，采用分栏布局：

```
┌─────────────────────────────────────┐
│  ← 设置                             │
├───────────┬─────────────────────────┤
│ 阅读体验  │                         │
│ 刷新设置  │  [当前分类的设置项]     │
│ 数据管理  │                         │
│ 快捷键    │                         │
│ 关于      │                         │
└───────────┴─────────────────────────┘
```

- 设置项变更**即时生效**，无"保存"按钮
- 危险操作（清空数据、重置）需二次确认弹窗
- "关于"页面显示：版本号、开源协议、反馈链接

## 默认值定义

```typescript
const DEFAULT_SETTINGS: Settings = {
  id: 'global',
  refreshInterval: 60,          // 1 小时
  maxArticlesPerFeed: 200,
  theme: 'system',
  fontSize: 'medium',
  markReadOnOpen: false,
  showUnreadOnly: false,
};
```
